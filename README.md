# WeatherStation

## External Unit

### Sensors
- Temperature
- CO2
- Humidity
- Pressure
- Wind Speed
- Rainfall
- UV Index

### Features
- SolarPower
- HomeAssistant integration (mqtt)

## Indoor Unit

### Sensors
- Temperature
- CO2
- Humidity
- Volatile Organic Compounds (VOC / TVOC)
- Pressure
- Particulate Matter (PM1.0 / PM2.5 / PM10)

### Features
- Power plug
- Display
- HomeAssistant integration (mqtt)
